# Personal Loan Campaign Targeting

### Business Goal
Identify high-probability loan buyers to improve targeting and campaign ROI for financial services.

### Tools & Methods
Python · Pandas · NumPy · Scikit-learn · Decision Trees · Model Evaluation · Business Insights

### Approach
- Cleaned and explored customer attributes
- Trained decision tree and compared to baseline
- Interpreted key drivers and provided business recommendations

### Result
Improved identification of likely responders, enabling more efficient marketing spend and better conversion.

### Next Steps / Improvement Ideas
- Compare more models (Random Forest, Gradient Boosting)
- Calibrate probabilities and create a lift chart
- Build a Power BI/Streamlit dashboard for stakeholders
